Overview Page

This Web Application has been created to meet Integrated Project 3 Module requirements for pass and gain satisfactiory mark.
We have been asked to design and develop a web application that access remote data sources via Application Programming Interfaces (APIs)
and presents fetched data using data visualization libraries or techniques in an interesting way within web browser.
​
Part of the project requires us to deliver:
  • Tutorial pages that explain where to source data
  • Aspects of data content and structure
  • API access
  • Visualization techniques used.
This project involves skills in:

  • Good research and analysis
  • Good knowledge of conceptualization
  • Creative design creation
  • Implementation and testing
  • Objective evaluation
  • Team working
  • Project management
  • Report writing and presentation skills
​ ​There are various stages of software development lifecycle implemented using the agile spiral methodology. This Web Application has been developed by carrying weekly sprints by all group members where product is developed through iterative line of software development beginning
with planning and ending on evaluation which is a start point for another sprint's planning stage.


Our product offers:

  • Access to real time data using Aplication Programming Interfaces.
  • Presenting fetched data by visualisation technics and libraries.
    • ​
  • Follow the most recent earthquakes of any world area but also query the results to display up to the last month of earthquakes and choose them based on their magnitudes.
  • Provides data for location of chosen earthquake.​
  • Display weather feed for chosen location with details of temperature, humidity and weather in a graphic.​
  • Signature count for petition “Revoke Article 50” in different Scottish constituencies.​
  • Display the air pollution level and details on a map over twelve years time.​
  • Chosen artist 10 top songs and ability to listen preview of songs.
  • Tutorial pages with implementation details and technical documentation for independent learning.
