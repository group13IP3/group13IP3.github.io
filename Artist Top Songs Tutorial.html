The Artist Top Songs Page Tutorial

About the artist top songs page

The artist top songs page allows users to search global artist top rated songs. A user needs to input the artist name into the text box and click the "Search" button that will populate the visualisation with the most and less popular songs for specefic artist. The visualisation has a form bubble graph and each bubble displays artist name, song title whith its rated and video in the next window.

The description of used technology

Ajax is an acronym that combines JavaScript and XML technology to communicate between the web browser (the client-site) and the server (the server site). The advantage of using Ajax is that this technology can request only data that needs to be updated, doesn’t required refreshing the page content. Thus, allow to respond quickly and makes website more interactive, dynamic and better user experience because of reducing connections to the server. Using this technology allows to write less coding for different web browsers.

Ajax can be used with different technology such as JQuery, JavaScript, HTML and CSS. JQuery makes available for use several methods for Ajax functionality. jQuery makes simpler the interaction with API by using some helpful methods like JQuery Ajax method. JQuery Ajax method can request and receive data in format – text, HTML, XML, JSON.

For this project, it was used jQuery Ajax() method.

The description of used technology for visualisation

D3.js is a JavaScript library for manipuleting documents based on data.This technology exposes the full capabilities of web standards such as HTML, SVG, and CSS. The advantage of D3 technology is extremely fast and supports large datasets, and dynamic behaviors for interaction and animation.
